# LC-3 Escape Room Game

## Description

This project is a text-based escape room game written in LC-3 assembly language.

The player explores an underground facility, collects items, restores power to the building, and attempts to escape through a locked exit door before triggering the facility security system.

The game includes:
- Multiple rooms
- Player movement
- Inventory system
- Interactive objects
- Win and loss conditions
- Code-entry puzzle system

---

## Controls

### Main Menu
- `1` - Move
- `2` - Interact
- `3` - Inventory
- `4` - Quit

### Movement
Use uppercase directions:
- `N` - North
- `S` - South
- `E` - East
- `W` - West

- LC-3 simulator capable of assembling and running `.asm` files
